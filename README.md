# Largest Element - Recursive Approach

This repository contains a C++ implementation of a recursive algorithm to find the maximum value in an array.

## 🚀 Concept
The algorithm uses a **divide and conquer** strategy via recursion:
- **Base Case:** If the array size is 1, the only element is the maximum.
- **Recursive Step:** Compare the last element with the maximum of the remaining $n-1$ elements.
